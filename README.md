# SVGoify


一个适用于前端的 svg 压缩、编辑工具


### 安装依赖

```shell
npm i
```

### 启动

```shell
npm run start
```

### 打包

生成可执行文件

```shell
npm run package
```

### 制作

根据 “生成可执行文件” 生成可分发文件，`npm run make` 包含了 `npm run package`

```shell
npm run make
```

### 分发


将安装程序分发到 GitHub 等平台（不在本地运行）

```shell
npm run publish
```
